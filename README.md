## Getting Started

First, install :

npm install @mui/material @emotion/react @emotion/styled

npm i vue-axios

npm i react-copy-to-clipboard

npm install react-router-dom

npm i qrcode.react

Second open phpmyadmin :
import shout url.sql

And run the development server:

npm run dev
